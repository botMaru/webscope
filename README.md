# WebScope (RP2350 Port)

WebScope is a web-based digital oscilloscope application designed for the Raspberry Pi Pico 2 (RP2350). It utilizes the WebUSB API to allow real-time signal visualization directly in the browser without any software installation.

This project is a fork of the original WebScope developed by Ing. Jan Fiala at the Czech Technical University in Prague. This specific version ports the core firmware to support the new RP2350.

This fork also finished some intended features in GUI menu. This updated version can be used at https://botmaru.github.io/webscope/

Original firmware author: Ing. Jan Fiala
